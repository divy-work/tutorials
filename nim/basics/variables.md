## Variables

```nim
var x = "string"
```

Or

```nim
var x: string= "string"
```

Similarly,
```nim
var an_array: seq[string] = @["smth", "whatever", "wow"]
# also nested lists:
var nested_array:seq[seq[string]] = @[@["r/hmm"]]
```

Both of them will ultimately return the same thing.
